# Algorithms 练习目录说明

本目录为 **ACM 模式**（`cin` / `cout`）的 C++ 题解，每个 `.cpp` 文件开头注释里有 **题目要点 + 算法思路 + 输入输出格式**。建议：**先看注释 → 遮住代码复述 → 对照实现**。

**编译（MinGW / g++）：**

```bash
g++ -std=c++17 -O2 某文件.cpp -o run.exe
```

**头文件：** 统一使用 `#include <bits/stdc++.h>`（MSVC 需自行改成标准头或换编译器）。

**代码结构：** 核心算法为 **与 LeetCode 同名的自由函数**（如 `lengthOfLongestSubstring`、`mergeTwoLists`），便于直接背诵与粘贴到题面；**不使用 `class Solution`**。设计类题目保留 **`LRUCache`**、手撕 **`DSU`** 等本体；`main` 仅负责读入、调用、输出。

---

## 文件命名

| 前缀 | 含义 |
|------|------|
| `lc####_...` | LeetCode 风格题号 + 英文短描述 |
| `lc####_..._2` | 同一题的 **另一种常考写法**（如递归/堆/DPS 等） |
| `extra_...` | **手撕基础**：排序、图论、字符串、数论等，不限于题号 |

## 一、排序与二分（先建立「有序」直觉）

| 主题 | 一句话 | 记忆点 | 文件 |
|------|--------|--------|------|
| 快排 | 选 pivot 划分，左右递归 | 面试最常写 Lomuto / Hoare | `extra_quick_sort.cpp` |
| 归并 | 二分两半排序再线性合并 | 稳定、O(n) 额外空间 | `extra_merge_sort.cpp` |
| 堆排 | 大根堆，堆顶与末尾交换 | 下标 `i` 孩子 `2i+1/2` | `extra_heap_sort.cpp` |
| 插入 | 前段有序，逐个插入 | 小数据、近乎有序快 | `extra_insertion_sort.cpp` |
| 计数 | 值域小非负整数 | O(n+k)，桶思想 | `extra_counting_sort.cpp` |
| 二分查找 | 有序数组找边界/存在性 | `mid = L+(R-L)/2` 防溢出 | `extra_binary_search.cpp` |
| 旋转数组 | 分段有序上二分 target | 比较 `nums[mid]` 与端点判哪半有序 | `lc0033_search_rotated_sorted_array.cpp` |
| 第 K 大 | 快选 / 大小为 k 的堆 | 快选 O(n) 期望；堆 O(n log k) | `lc0215_kth_largest_element.cpp`、`_2.cpp` |

**口诀：** 划分用快排，稳定用归并，原地堆排，值域小计数；有序想二分，第 K 想堆或快选。

## 建议学习顺序（由易到难）

1. **排序 + 二分 + Kadane**（建立复杂度与边界意识）  
2. **链表反转 / 合并**（指针不绕）  
3. **滑窗、双指针、区间合并**  
4. **栈（括号）、简单 BFS（层序、网格）**  
5. **回溯（排列）、堆/快选、LIS**  
6. **图：拓扑、最短路、并查集**  
7. **KMP、LRU、链表综合（K 组翻、重排）**
